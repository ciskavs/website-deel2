Ontwerp thema's

Het schrijven van CSS-regels voor elementen zoals section en p is geweldig, maar wat als je wil dat sommige er anders uitzien dan andere? Hier leer je hoe je verschillende stijlregels toepast op elementen van hetzelfde type en een ander thema maakt voor elke pagina op je website!

  • Ga naar je stylesheet bestand styles.css en voeg het volgende toe - zorg ervoor dat je de punt vooraan plaatst!
.lijnErboven {
border-top-style: solid;
 border-top-width: 2px;
 border-top-color: #F5FFFA;
 padding-bottom: 10px;
 }
  • Ga nu naar attracties.html (of het HTML-bestand waaraan je werkt als je je eigen project gebruikt) en voeg het volgende attribuut toe aan elke section tag:
<section class="lijnErboven">

Je zou boven elke sectie op de pagina een lijn moeten zien verschijnen. Gefeliciteerd - je hebt zojuist je eerste CSS-class gebruikt!


  • Kijk hoe je webpagina er nu uitziet en vergelijk deze met de andere pagina's met section tags. Je zal zien dat alleen de regels waaraan je het attribuut class="lijnErboven" hebt toegevoegd, de lijn bovenaan hebben.

Onthoud dat als je een CSS selector zoals section of p of nav ul gebruikt, de stijlregels van toepassing zijn op alle elementen van dat type op je website.

Met CSS class kun je de stijl van enkele elementen wijzigen.

Door een punt vóór de selector te zetten, wordt het een class selector. Een class kan elke naam hebben, dus het hoeft niet de naam van een HTML-element te zijn. Bijvoorbeeld:

.mijnToffeClass {
/* mijn toffe stijlregels komen hier! */
}

Om aan te geven op welke elementen de stijlregels van toepassing zijn, voeg je het class attribuut toe aan de elementen in de HTML-code: typ de naam van de class in als de waarde voor het attribuut, zonder de punt, zoals hieronder:

class="mijnToffeClass"
  • Zullen we nog een class toevoegen? Pas de volgende CSS-code aan in styles.css:
.vetKader {
background-color: #87CEFA;
 color: #A52A2A;
 border-style: solid;
 border-width: 2px;
 border-color: #F5FFFA;
 border-radius: 10px;
 }
  • Voeg nu op een andere pagina van je website de class toe aan een paar elementen. In het voorbeeld voegen we het toe aan de section tags op de 'Eten in Leiden' pagina van de website, als volgt: <section class="vetKader">.

Het ziet er vet uit, maar nu zijn de secties allemaal samengeplet.


Je kan zoveel CSS classes aan een element toevoegen als je maar wil. Schrijf gewoon de namen van alle klassen die je wil gebruiken binnen het class attribuut (vergeet niet: zonder de punt!), met spaties ertussen.

  • Laten we nog een CSS class maken om de secties wat ruimte te geven. Maak in het bestand styles.css de volgende CSS class aan:
.beetjeRuimte {
padding: 10px;
 margin-top: 20px;
 }
  • Voeg in je html code de nieuwe class toe aan elk van de section tags waar je aan werkte, zoals dit:
<section class="vetKader beetjeRuimte">

Met CSS classes kun je dus kiezen welke elementen je wilt opmaken en kun je dezelfde set stijlregels opnieuw gebruiken voor alle elementen die je wilt.

  • Ga naar index.html en voeg de klasse vetKader toe aan het main element of een ander element op de pagina. Je kunt het later weer verwijderen!
<main class="vetKader">

Zo ziet de home pagina eruit met de CSS class. De class lijnErboven is ook aan de img tag toegevoegd met de afbeelding van Tito.


Uitdaging: maak zelf nieuwe classes

Gebruik CSS classes om een paar verschillende formaten voor je afbeeldingen te maken, bijvoorbeeld .kleineAfbeelding en .grotereAfbeelding. Haal vervolgens het attribuut width van elk van de img tags weg en voeg in plaats daarvan de juiste class toe.

Je kan een CSS class maken die alleen de width (breedte) van een element definieert, zoals dit:

.kleineAfbeelding {
width: 100px;
 }

Zo ziet de img tag met het width attribuut eruit:

<img src="tito.png" alt="Tito de hond" width="100px" />

Als je het width attribuut weghaalt en de grootte van de afbeelding bepaalt met de CSS klasse, ziet het er zo uit:

<img src="tito.png" class="kleineAfbeelding" alt="Tito de hond" />

Door een CSS class te gebruiken kan je gemakkelijk de breedte van alle afbeeldingen in één keer veranderen. Je hoeft hiervoor maar één regel code te veranderen in je stijlregels!
