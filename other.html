Code van een andere website

Opmerking: voor deze stap heb je Chrome, Firefox of Internet Explorer/Edge nodig. Als je geen van deze browsers hebt, kun je doorgaan met de volgende stap 'Grootte aanpassen'.

Hier leer je hoe je een kijkje kunt nemen in de code van een website met behulp van de inspector tool (pagina inspecteur). Je leert ook hoe je wijzigingen kan aanbrengen die alleen jij kan zien!

  • Zorg voordat je begint dat je project is opgeslagen. Vernieuw vervolgens je website door op het verversen pictogram in je browser te klikken. Dit 'pijl-in-cirkel' pictogram vind je links naast het adres van de website.
  • Markeer op je webpagina (de eigenlijke pagina, niet de code) de tekst 'Mijn website gaat over Leiden', klik er met de rechtermuisknop op en selecteer de optie Inspect in het menu dat verschijnt. (De optie kan 'element controleren' of 'element inspecteren' of iets dergelijks worden genoemd, afhankelijk van welke browser je gebruikt. Als je deze menuoptie niet kunt vinden, vraag dan aan een van de mentoren om hulp.)

Een nieuw venster verschijnt in je webbrowser met veel tabbladen en code: de developer tools (ontwikkelaarstools) of kortweg dev-tools. Hier kan je de code zien voor het item waarop je hebt geklikt, evenals de code voor de hele pagina!

De HTML-code inspecteren

  • Zoek naar het tabblad met de HTML-code voor de pagina (dit kan 'Elementen' of 'Inspector' worden genoemd). De code moet er ongeveer hetzelfde uitzien als hoe je hem in je HTML-bestand hebt getypt! Je kan op de kleine driehoekjes klikken om de verborgen code uit te vouwen.
  • Dubbelklik op de tekst tussen de tags. Je zou het nu moeten kunnen bewerken! Typ iets in en druk op Enter.
  • Zie je dat de tekst op je website is aangepast? Opmerking: alleen jij kan deze wijzigingen zien.
  • Laad nu de pagina opnieuw en kijk wat er gebeurt. Je wijzigingen moeten verdwijnen!
  • Klik in de linkerbovenhoek van het vak dev-tools op het pictogram dat lijkt op een klein vierkant met een pijl. Nu kan je de cursor over de webpagina bewegen en de HTML-inspecteur toont je de code die deze beschrijft.

De CSS-code inspecteren

  • Laten we de CSS-code eens bekijken. Zoek naar het tabblad Stijlen in de dev-tools (dit kan 'Stijleditor', 'Styles' of iets dergelijks worden genoemd). Je zou een aantal CSS-regels moeten zien, inclusief degenen die je voor die paragraaf hebt gemaakt, zoals #mijnCooleTekst.
  • Klik in de regels #mijnCooleTekst op de waarde naast de eigenschap color. Probeer een andere waarde in te typen. Kijk hoe de tekst op je webpagina van kleur verandert!

Opmerking: je kan ook op het gekleurde vierkant klikken om de kleur te wijzigen met behulp van een kleurenkiezer.

  • Klik in de ruimte achter de kleur. Er begint een nieuwe regel, waarin je meer CSS kunt typen. Typ het volgende en druk op Enter:
background-color: #660066;

Als het goed is, is de achtergrond van dat stuk tekst veranderd.


Als je de websitecode wijzigt met behulp van de dev-tools, wijzig je tijdelijk hoe het in je browser eruit ziet. Je verandert eigenlijk niet de bestanden waaruit de website bestaat.

Als je de pagina vernieuwt, laad je de website opnieuw vanuit de bestanden (op internet of op je computer). Dat is waarom je veranderingen verdwijnen.

Nu je dat weet, kun je lekker rotzooien met de code op andere websites!

  • Probeer deze tools te gebruiken om de code op een andere website te bekijken. Je kunt zelfs wijzigingen aanbrengen als je wilt! Vergeet niet dat alleen jij de wijzigingen kan zien die je aanbrengt en dat alles wordt gereset wanneer je de pagina vernieuwt.
