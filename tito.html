Leer Tito omrollen!

Je kan je website interactiever maken door coole dingen te doen als je met de muiscursor over de dingen beweegt ('hover' in het Engels)!

  • Zoek de CSS-regels voor de img tags of maak er enkele als je er geen hebt. Voeg een rand toe en voeg direct eronder een nieuwe regels toe:
img {
border: 2px solid White;
 }
img:hover {
border: 2px dashed Navy;
 }

Je hebt zojuist een speciaal type CSS-blok gebruikt dat een pseudo-class wordt genoemd.

Een pseudo-class is een beetje anders dan een class die je zelf maakt. Je kunt het herkennen aan de :.

Pseudo-classes zijn ingebouwd in HTML-elementen: je kan :hover stijlregels toevoegen aan elk element, class of id selector in je stylesheet zonder dat je iets extra's in je HTML-code hoeft toe te voegen.

  • Wat denk je dat er gaat gebeuren? Ga naar een pagina op je website waar een afbeelding staat (voeg een afbeelding toe als die er niet zijn!) en beweeg je cursor over een afbeelding om erachter te komen!
  • Laten we deze nieuwe :hover pseudo-class samen met een CSS-class gebruiken om links (koppelingen) van kleur te laten veranderen als je eroverheen beweegt! Voeg een link toe aan je webpagina en voeg een attribuut toe om de class name op te geven. Houd er rekening mee dat links worden gedefinieerd met de tag <a>, zoals:
<p>
Bezoek de <a class="leukeLinks" href="https://nl.wikipedia.org/wiki/Leiden">Wikipedia pagina</a> om meer te leren over Leiden!
 </p>
  • Voeg de volgende code toe aan je stylesheet en voer vervolgens de code uit om je leuke links in actie te zien.
.leukeLinks {
text-decoration: none;
 color: #FFFAF0;
 }
.leukeLinks:hover {
color: #00FF7F;
 }
  • Laten we het attribuut class="leukeLinks" toevoegen aan alle links in de menubalk.

Je kan al deze trucs ook combineren met animaties!

  • Ga weer naar het CSS-blok voor de afbeelding van Tito (of welke afbeelding je eerder aan het bewerken was). Voeg de volgende code toe aan je stylesheet-bestand:
#fotoTito {
border-radius: 100%;
 width: 100px;
 }
#fotoTito:hover {
animation-name: omrollen;
 animation-duration: 1s;
 animation-iteration-count: 1;
 }
@keyframes omrollen {
0% {
transform: rotate(0deg);
 }
 100% {
transform: rotate(-360deg);
 }
 }
  • Kun je raden wat er zal gebeuren?
  • Roep nu "Omrollen!" Terwijl je de cursor over de afbeelding beweegt!

Uitdaging: van kleur veranderende regenboog links maken

Kun je de regenboogGloed animatie van de vorige stap gebruiken om de kleuren van de links in je menu te laten veranderen als de cursor eroverheen zweeft?

Hieronder staat de code voor de regenboogGloed animatie. Er zijn vijf fases gedefinieerd en in elke fase wordt een andere tekstkleur ingesteld. Je kan meer toevoegen of wijzigen als je wil!

@keyframes regenboogGloed {
0% {
color: #00BFFF;
 }
25% {
color: #00FF7F;
 }
50% {
color: #eeeeaf;
 }
75% {
color: #eeafee;
 }
100% {
color: #00BFFF;
 }
}

Als je iets wilt animeren, voeg je de drie animation eigenschappen toe aan de stijlregels, zoals je eerder hebt gedaan. Zorg er altijd voor dat de animation-name overeenkomt met de naam van de animatie die je wil gebruiken.


Je kan hover effecten op deze manier aan je nav menu toevoegen:

nav ul li a:hover{
animation-name: regenboogGloed;
 animation-duration: 1.5s;
 animation-iteration-count: infinite;
 }

Of, als je wil dat andere links op je website ook veranderen in regenboogkleuren, kan je in plaats daarvan de animatie toevoegen aan de class .leukeLinks:

.leukeLinks:hover{
color: #00BFFF;
 animation-name: regenboogGloed;
 animation-duration: 1.5s;
 animation-iteration-count: infinite;
 }
